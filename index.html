📖 API Documentation

Use your API key via the X-API-Key header or ?api_key= query parameter.

GET /api/v1/policies
List all policies (paginated). Params: limit, offset
GET /api/v1/policies/search?q=...&field=policyno
Search by policyno, name, or both
GET /api/v1/policies/{policyno}
Get a single policy by exact policy number
GET /api/v1/policies/count
Get total record count

💻 Example Usage

curl -H "X-API-Key: YOUR_KEY" https://your-domain.com/api/v1/policies?limit=10

curl "https://your-domain.com/api/v1/policies/search?q=JOHN&field=name&api_key=YOUR_KEY"

curl -H "X-API-Key: YOUR_KEY" https://your-domain.com/api/v1/policies/123456789

Fields returned: sn, policyno, name, doc, plan, mode, premium, sumass, fup, mobileno, status
Note: Status values "due" and "paid" are returned as blank.
